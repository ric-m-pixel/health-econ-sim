# Health Technology Assessment (HTA) Decision Suite

A live, interactive Monte Carlo simulation tool for evaluating the cost-effectiveness of clinical interventions.

### Key Features:
- **Probabilistic Sensitivity Analysis (PSA):** 10,000 simulations per run.
- **Cost-Effectiveness (CE) Plane:** Visualizing uncertainty and WTP thresholds.
- **Tornado Analysis:** Identifying key drivers of economic value.
- **Switchable Models:** Supports both Clinical Success and QALY-based HTA frameworks.

**Live App:** health-sim.streamlit.app
